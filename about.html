About

This site is a place where you can find games made on Orteil's Idle Game Maker. The games mainly come from the DashNet Server, r/idlegamemaker, the discontinued forums, and possibly other places if applicable, which are then placed here for easy finding.

If you have an issue, perhaps join the discord server? https://discord.gg/BTwyDHBWey

If you prefer email, here you go. ericrylie at gmail dot com

You can also see the code for this place here.

Why?

Prior to Discord introducing forum channels, the DashNet channel for IGM could only pin 50 games (it's less than that since there were other pinned messages). Someone decided to combat this by making IGM games lists. That someone is the person writing this. They spend too much time doing this.

This was originally a part of the IGM Fan Guide, but got moved to the r/idlegamemaker subreddit before coming here! If you're curious on the bigger history of it all, check out the the reddit post or my personal site post about it.

Who even made this place?

This place is managed by AgentRylie. She has been part of the community since 2019, and has been doing stupid lists like this since 2020. She tends to create games with some lore in them, as well as writings like this page you are reading right now. In her free time... she does the same thing because that's basically the only time she could do stuff like that.

If there is no writer credit on a page, it is safe to assume Rylie wrote it.

You can see her games here.

If you are curious, here's my setup: Notepad++ for site files, GitHub Desktop to push changes, PaintNet to draw banners, Firefox and/or Microsoft Edge to test web pages, and Google Sheets to keep track of the games list.
